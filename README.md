## GitHub Profile Explorer


### Features
- Search any GitHub username
- View profile, bio, followers, repos
- Star repos locally (localStorage)
- Rate limit handled with friendly error message

### What I learned
- React Router for multi-page navigation with URL params
- TanStack Query replaces useEffect+useState for fetching —
  handles loading, error, and caching automatically
- Code splitting with lazy() loads each page on demand
  instead of one giant JS bundle
